# **App Name**: NutritionWise

## Core Features:

- CSV Import: Import food data from a CSV file.
- Food List Display: Display foods in a paginated list (up to 8 per page) with nutritional information.
- Search: Search for foods by name.
- Meal Creation: Create meals by selecting foods from the list and specifying quantities in grams.
- Nutritional Calculation: Calculate total nutritional values (calories, carbs, proteins, fats, etc.) for each meal.
- Favorites: Save favorite foods and meals.
- Settings: Configure app parameters, such as the maximum number of foods displayed per page.

## Style Guidelines:

- Primary color: Light green (#A7D1AB) to evoke health and nature.
- Background color: Very light green (#E9F5EB), nearly white.
- Accent color: Muted orange (#D1B4A7) for highlights and call-to-action buttons.
- Font pairing: 'Inter' (sans-serif) for body and 'PT Sans' (sans-serif) for headlines.
- Use clear and modern icons for food categories and meal actions.
- Bottom navigation bar for main actions (food list, meal creation, favorites, settings).
- Subtle animations for adding/removing foods from meals and saving favorites.
